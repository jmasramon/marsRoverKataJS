## Getting Started

Install the module with: `npm install marsrover`

```js
var marsrover = require('marsrover');
marsrover.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g marsrover
$ marsrover --help
$ marsrover --version
```
